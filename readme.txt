Penguinは、個人的なクロッキーの練習のために作られました。

動作環境:
  Windows10

機能一覧:
1. 画像のランダム表示:
   指定したフォルダの画像をランダムに表示します。

2. 時間/回数の設定
   00:00:00～23:59:00の間で時間を、1～99の間で回数を設定できます。

3. 履歴
   直近、10枚まで、表示した画像を確認できます。

使用方法:
1. 右クリックでメニューを開きます。
   設定→フォルダからフォルダブラウザが開くので、
   表示したい画像が入っているフォルダを指定します。

2. 右クリックでメニューを開きます。
   設定→時間/回数から、時間と回数を指定します。
   この手順は、スキップ可能で、デフォルトは、30秒x10回です。

3. 右クリックでメニューを開きます。
   開始をクリックして、クロッキーを開始します。

終了方法:
1. 右クリックでメニューを開きます。
   終了をクリックして、Penguinを閉じます。

ショートカットキー:

Ctrl + O : フォルダブラウザを開きます。
Ctrl + P : クロッキーを開始します。
Ctrl + Q : Penguinを終了します。
Space : タイマーを一時停止/再開します。
Ctrl + 中ボタン : ウィンドウを移動します。
